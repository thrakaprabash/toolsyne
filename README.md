# Toolsyne

Toolsyne is a fast, user-friendly, and free online tool hub featuring an extensive collection of calculators, converters, and reference tools. 

## Structure

The site has been fully migrated to a modern React setup using Vite. You can find the main application code inside the `modern_app/` directory.

## Running Locally

1. Navigate to the application directory:
   ```bash
   cd modern_app
   ```
2. Install dependencies (if you haven't already):
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## Deployment

The project is optimized for deployment on Netlify. You can deploy by connecting this repository to Netlify via Git, or by building the `modern_app` directory locally to the `dist` folder and deploying it manually.

```bash
# Build command
cd modern_app
npm run build
```
