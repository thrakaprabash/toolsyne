How to Use GPA Calculator

  1. Select the number of courses you want to calculate
  2. Enter each course name (optional)
  3. Enter the credit hours for each course
  4. Enter your grade (letter or percentage)
  5. Click "Calculate GPA" to see your result
  6. Use the CGPA converter tab to convert between scales

Grading Scale Reference

  • A+ (97-100%) = 4.0
  • A (93-96%) = 4.0
  • A- (90-92%) = 3.7
  • B+ (87-89%) = 3.3
  • B (83-86%) = 3.0
  • B- (80-82%) = 2.7
  • C+ (77-79%) = 2.3
  • C (73-76%) = 2.0
  • C- (70-72%) = 1.7
  • D+ (67-69%) = 1.3
  • D (60-66%) = 1.0
  • F (Below 60%) = 0.0
